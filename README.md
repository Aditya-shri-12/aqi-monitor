# AirAware AI Dashboard

A professional, real-time Air Quality Index (AQI) monitoring application with AI-powered analysis.

## Features

- ✅ **Real-time AQI Data** - Uses free public APIs (no API key required!)
- ✅ **City Validation** - Validates city names before fetching data
- ✅ **Professional Error Handling** - Clear error messages for invalid cities
- ✅ **AI-Powered Analysis** - Intelligent health recommendations based on AQI levels
- ✅ **Beautiful UI** - Modern glassmorphism design with dynamic backgrounds
- ✅ **Responsive Design** - Works on desktop and mobile devices
- ✅ **100% Free** - No API keys, no registration, no setup required!

## Setup Instructions

### No Setup Required!

This app uses completely free public APIs that require no API keys:
- **OpenStreetMap Nominatim** - For city geocoding and validation
- **Open-Meteo** - For weather and air quality data

Simply open `AirAware AI Dashboard.html` in your web browser and start using it!

## Usage

1. Enter a city name in the search box (e.g., "New York", "Delhi", "London")
2. Click the search button or press Enter
3. View real-time AQI data, weather information, and AI-powered health recommendations

## Error Handling

The app will show clear error messages if:
- City name is invalid or doesn't exist
- API key is not configured
- Network connection issues occur
- Air quality data is unavailable for the location

## Technologies Used

- HTML5
- CSS3 (with Tailwind CSS)
- JavaScript (ES6+)
- Chart.js for data visualization
- Font Awesome icons
- OpenWeatherMap API

## API Endpoints Used

- **OpenStreetMap Nominatim API**: For city geocoding and validation (free, no API key)
- **Open-Meteo Weather API**: For real-time weather data (free, no API key)
- **Open-Meteo Air Quality API**: For real-time AQI and pollutant data (free, no API key)

All APIs are completely free and require no registration or API keys!
